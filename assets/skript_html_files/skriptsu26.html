1.9.1  Die algebraische Struktur: ℝ ist ein Körper.

Ein Körper ist eine Menge (in unseren Fall ℝ), auf welcher eine Addition und eine Multiplikation + : ℝ × ℝ → ℝ d.h. (x,y)↦ +x + y, ⋅ : ℝ × ℝ → ℝ d.h. (x,y)↦⋅x ⋅ y

definiert sind, welche folgende Eigenschaften erfüllen: x + y = y + x Kommutativität x ⋅ y = y ⋅ x x + (y + z) = (x + y) + z Assoziativität x ⋅ (y ⋅ z) = (x ⋅ y) ⋅ z ∃0∈ℝ∀x∈ℝ0 + x = x neutrales  Element ∃1∈ℝ∀x∈ℝ1 ⋅ x = x ∀x∈ℝ∃(−x)∈ℝx + (−x) = 0 inverses  Element ∀x∈ℝ\{0}∃x−1∈ℝx ⋅ x−1 = 1 x ⋅ (y + z) = x ⋅ y + x ⋅ z Distributivität

Diese Beziehungen folgen elementar über die Definition von ℝ aus den entsprechenden Eigenschaften der rationalen Zahlen.
