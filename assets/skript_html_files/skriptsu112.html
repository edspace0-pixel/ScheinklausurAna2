3.1.2  Ableitungen in reellen und komplexen Variablen

Entsprechend der Folgendefinition des Grenzwertes von Funktionen ist für Funktionen einer reellen Veränderlichen die Konvergenz in (3.1) an {φ(f; hk,x)}k=1∞ für beliebige Folgen reeller hk → 0 zu verifizieren, für Funktionen einer komplexen Veränderlichen muss man hingegen beliebige Folgen komplexer hk → 0 betrachten.

Es sei nun f : X → K2n wobei X eine offene Teilmenge von ℂ ist. Wir betrachten einen Punkt x0 ∈ X ∩ ℝ. Da die Menge XR = X ∩ ℝ eine offene Menge in ℝ ist (Beweisen Sie dies!), so ist x0 sowohl ein innerer Punkt von X bezüglich der Metrik in ℂ als auch ein innerer Punkt von XR bezüglich der Metrik in ℝ. Wir können dann neben der Funktion f einer komplexen Veränderlichen auch deren Einschränkung f|ℝ auf reelle Argumente betrachten und damit folgende zwei Ableitungen diskutieren (ℂ) − f′(x 0) = lim h∈ℂ h→0φ(f; h,x0), (3.2) (ℝ) − f′(x 0) = lim h∈ℝ h→0φ(f|ℝ; h,x0). (3.3)

Der Grenzwert (3.2) entspricht der Ableitung 3.1 von f als Funktion einer komplexen Variablen und wird komplexe Ableitung genannt. Existiert diese, so heisst f im Punkt x0 komplex differenzierbar. Der Grenzwert (3.3) entspricht der Ableitung 3.1 von f|ℝ als Funktion einer reellen Variablen und wird reelle Ableitung genannt. Existiert dieser, so heisst f im Punkt x0 reell differenzierbar.

Es gilt folgender Satz:

Theorem 3.1.2. Ist die Funktion f im Punkt x0 ∈ X ∩ ℝ komplex differenzierbar, so ist diese Funktion in x0 auch reell differenzierbar. Die komplexe und die reelle Ableitung stimmen dabei überein.

Beweis. Die Menge der komplexen Folgen {hn}→ 0 enthält als Teilmenge die Menge der reellen Folgen {hn}→ 0. Der Satz folgt damit sofort aus der Folgendefinition des Grenzwertes von Funktionen. □

Example 3.1.3. Es sei f ℂ → ℂ mit f(x) = x = const. Dann gilt

φ(f; h,x) = 1 h(c − c) = 0,h ∈ ℂ,h≠0,

und damit f′(x) = 0 für x ∈ ℂ. Falls x ∈ ℝ, so gilt damit stets auch (ℂ) − f′(x) = (ℝ) − f′(x) = 0.

 

Example 3.1.4. Es sei f : ℂ → ℂ mit f(x) = x. Dann gilt

φ(f; h,x) = 1 h((x + h) − x) = h h = 1,h ∈ ℂ,h≠0,

und damit f′(x) = 1 für beliebige x ∈ ℂ (und damit auch als komplexe oder reelle Ableitung in ℝ).

 

Example 3.1.5. Es sei f : ℂ \{0}→ ℂ mit f(x) = 1 x für x≠0. Dann gilt φ(f; h,x) = 1 h 1 x + h −1 x = 1 h x − (x + h) x(x + h) = − 1 x(x + h) →− 1 x2,h → 0,h ∈ ℂ.

und damit f′(x) = −1 x2 für x≠0, x ∈ ℂ (und damit auch als komplexe oder reelle Ableitung in ℝ).

Problem 3.1.6. Berechnen Sie auf gleichem Wege die Ableitung von f(x) = xm, m ∈ ℤ, x ∈ ℤ, sowie x≠0 für m < 0.

Wir diskutieren jetzt Beispiele, bei denen die reelle Ableitung existiert, die komplexe Ableitung allerdings nicht.

Example 3.1.7. Es sei f : ℂ → ℂ, f(x) = x¯. Da f|ℝ(x) = x¯ = x, x ∈ ℝ, so ist f in ℝ reell differenzierbar und (ℝ) − f′(x) = 1 für x ∈ ℝ. Auf der anderen Seite ist f in keinem Punkt x ∈ ℂ komplex differenzierbar. Tatschlich, da

φ(f; h,x) = 1 h(x + h¯ −x¯) = h¯h−1,h ∈ ℂ,h≠0.

Wählt man nun hn(1) = n−1 → 0, so gilt {φ(f; hn(1),x)} n=1∞ = {1} n=1∞ → 1. Für hn(2) = in−1 → 0 gilt hingegen {φ(f; hn(2),x)} n=1∞ = {−1} n=1∞ →−1. Damit existiert der Grenzwert lim h→0;h∈ℂφ(f; h,x) nicht, also ist f nicht komplex differenzierbar.

 

Example 3.1.8. Es sei f : ℂ → ℝ, f(x) = |x|2 = xx¯. Für x ∈ ℝ gilt f|ℝ(x) = x2. Damit ist f auf ℝ reell differenzierbar mit (ℝ) − f′(x) = 2x, x ∈ ℝ. Aus folgendem Satz folgt, dass die Ableitung von f als Funktion einer komplexen Variablen hingegen in keinem einzigen Punkt x ∈ ℂ existiert:

Theorem 3.1.9. Die Menge X sei eine offene Teilmenge von ℂ sowie f : X → ℝn, n ∈ ℕ und x0 ∈ X. Ist f im Punkt x0 als Funktion einer komplexen Variablen differenzierbar, so gilt f′(x 0) = 0.

Beweis. Wir betrachten zunächst den Fall n = 1. Da f nur reelle Werte annimmt, so ist

φ(f; h,x0) = h−1(f(x 0 + h) − f(x0)︸ ∈ℝ),h ∈ ℂ,h≠0,

reell für reelle h und rein imaginär für Reh = 0. Da f differenzierbar ist, so existiert der Grenzwert f′(x 0) = lim h→0;h∈ℂφ(f; h,x0). Betrachtet man eine Folge reeller hn≠0, hn → 0, so ergibt sich nach der Folgendefinition

Imf′(x 0) = lim hn→0;hn∈ℝφ(f; hn,x0) = 0.

Für eine Folge rein imaginärer Werte hn≠0, ihn ∈ ℝ, hn → 0, erhält man zudem

Ref′(x 0) = lim hn→0;ihn∈ℝφ(f; hn,x0) = 0.

Damit ist f′(x 0) = 0. Den Fall n ≥ 1 löst man durch Betrachtung der einzelnen Komponenten πk(f(x)). □

Problem 3.1.10. Beweisen Sie unter Anwendung von Satz 3.1.9, dass die Funktion f(x) = |x|2 in keinem Punkt x ∈ ℂ als Funktion einer komplexen Variablen differenzierbar ist.
