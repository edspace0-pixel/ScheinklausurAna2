2.7.4  Ein Vergleich der Strukturen von ℂn und ℝ2n.

So wie sich eine komplexe Zahl z = x + iy = (x,y) als Paar reeller Zahlen darstellen lässt, so kann man einen Vektor z ∈ ℂn gegeben durch

z = (z1,…,zn) = (x1 + iy1,…,xn + iyn)

mit dem Vektor

ρ = (x1,y1,x2,yn,…,xn,yn) ∈ ℝ2n

identifizieren. Diese Abbildung zwischen ℂn und ℝ2n ist bijektiv und respektiert offensichtlich die Operation der Addition in beiden Räumen, denn z′ + z″ = (z 1′ + z 1″,…,z n′ + z n″) = (x1′ + x 1′ + i(y 1′ + y 1″),…,x n′ + x n″ + i(y n′ + y n″))

in ℂn entspricht ja in ℝ2n dem Vektor

(x1′ + x 1″,y 1′ + y 1″,…,x n′ + x n″,y n′ + y n″) = ρ′ + ρ″.

Ebenso berechnet sich die Norm in beiden Räumen gleich zu

∥ z ∥2 = ∑ k=1n|z k|2 = ∑ k=1n(x k2 + y k2) =∥ ρ ∥2.

Man nennt deshalb ℂn und ℝ2n bezüglich der Addition und der Norm isomorph.
