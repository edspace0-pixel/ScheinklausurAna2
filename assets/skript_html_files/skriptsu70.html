2.7.2  Die Hermitsche Struktur - das komplexe Skalarprodukt.

Es sei X ein komplexer linearer Vektorraum. Ein komplexes Skalarprodukt ist eine Abbildung

⋅,⋅ : X × X → ℂ

mit folgenden Eigenschaften:

x,y = y,x¯, α′x′ + α″x″,y =α′x′,y + α″ x″,y, x,x ≥ 0 ∧ x,x = 0 ⇔x = 0 ∈ X (2.28)

für beliebige x,y,x′,x″ ∈ X und α′,α″ ∈ ℂ. Aus der ersten und zweiten Identität von (2.28) folgt

x,β′y′ + β″y″ = β′¯x,y′ + β″¯ x,y″

für beliebige x,y′,y″ ∈ X und β′,β″ ∈ ℂ: Das komplexe Skalarprodukt ist linear im ersten und antilinear im zweiten Argument. Für X = ℂn und

x = (x1,…,xn) ∈ ℂn,y = (y 1,…,yn) ∈ ℂn,

gibt es folgende kanonische Wahl des Skalarproduktes

x,y := x1y1¯ + ⋯ + xnyn¯. (2.29)

Problem 2.7.1. Verifizieren Sie die Eigenschaften (2.28) für (2.29).
