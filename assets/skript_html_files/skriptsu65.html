2.4.5  Skizze des Beweises von Satz 2.4.5.

Problem 2.4.6. Vervollständigen Sie den Beweis selbständig nach folgendem Plan:

Schritt 1: Für

yn = 1 + 1 nn = 1 + n 1 1 n + ⋯ + n r 1 nr + ⋯ + 1 nn.

folgt wegen

n r 1 nr = n(n − 1)⋯(n − r + 1) r! ⋅ 1 nr < 1 r!

die Abschätzung yn < xn < e für n ∈ ℕ.

Schritt 2: Die Folge {yn}n∈ℕ ist monoton wachsend. Wegen Schritt 1 besitzt diese Folge damit einen Grenzwert g = lim n→∞yn wobei g ≤ e.

Schritt 3: Für alle n ∈ ℕ gibt es ein m(n) ∈ ℕ so dass ym(n) > xn. Daraus folgt g = e.
