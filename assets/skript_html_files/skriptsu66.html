2.6.1  Die Struktur des reellen linearen Vektorraumes.

Eine Menge X nennt man einen reellen linearen Vektorraum, wenn auf X eine Addition + : X × X → X sowie die Multiplikation von Elementen aus X mit reellen Zahlen ⋅ : ℝ × X → X definiert sind, welche folgende Eigenschaften erfüllen:

∀x,y∈X x + y =y + x, ∀x,y,z∈X (x + y) + z =x + (y + z), ∃0∈X∀x∈X x + 0 =x, ∀x∈X∃−x∈X x + (−x) =0, ∀α,β∈ℝ α ⋅ (β ⋅ x) =(αβ) ⋅ x, ∀x∈X 1 ⋅ x =x, ∀α,β∈ℝ∀x∈X (α + β) ⋅ x =(α ⋅ x) + (β ⋅ x), ∀α∈ℝ∀x,y∈X α ⋅ (x + y) =(α ⋅ x) + (α ⋅ y). (2.22)
