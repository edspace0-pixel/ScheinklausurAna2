2.6.2  Die Euklidsche Struktur - das reelle Skalarprodukt.

Es sei X ein reeller linearer Vektorraum. Ein reelles Skalarprodukt ist eine Abbildung

⋅,⋅ : X × X → ℝ

mit folgenden Eigenschaften:

x,y = y,x, α′x′ + α″x″,y =α′x′,y + α″ x″,y, x,x ≥ 0 ∧ x,x = 0 ⇔x = 0 ∈ X (2.23)

für beliebige x,y,x′,x″ ∈ X und α′,α″ ∈ ℝ. Aus der ersten und zweiten Identität von (2.23) folgt

x,β′y′ + β″y″ = β′x,y′ + β″ x,y″

für beliebige x,y′,y″ ∈ X und β′,β″ ∈ ℝ: Das reelle Skalarprodukt ist linear im ersten und im zweiten Argument. Für X = ℝn und

x = (x1,…,xn) ∈ ℝn,y = (y 1,…,yn) ∈ ℝn,

gibt es folgende kanonische Wahl des Skalarproduktes

x,y := x1y1 + ⋯ + xnyn. (2.24)

Problem 2.6.1. Verifizieren Sie die Eigenschaften (2.23) für (2.24).
