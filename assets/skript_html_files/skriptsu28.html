1.9.3  Die topologische Struktur: Die Menge ℝ genügt dem Intervallschachtelungsaxiom (Vollständigkeitsaxiom).

Für an,bn ∈ ℝ sei

In = [an,bn] = {x ∈ ℝ|an ≤ x ≤ bn},n ∈ ℕ.

Es gelte zudem

∀n∈ℕan ≤ an+1 ≤ bn+1 ≤ bn.

Daraus folgt ∩n∈ℕIn = ∩n∈ℕ[an,bn]≠∅, d.h. dieser Durchschnitt enthält zumindest eine reelle Zahl. Diese Eigenschaft, die sogenannte Vollständigkeit von ℝ wird im weiteren Verlauf der Vorlesung bewiesen.
