2.1.1  Intervalle und ε-Umgebungen.

Für zwei reelle Zahlen a,b ∈ ℝ definieren wir das offeneIntervall]a,b[ = def {x ∈ ℝ|a < x < b}, halboffeneIntervall]a,b] = def {x ∈ ℝ|a < x ≤ b}, halboffeneIntervall[a,b[ = def {x ∈ ℝ|a ≤ x < b}, abgeschlosseneIntervall[a,b] = def {x ∈ ℝ|a ≤ x ≤ b}.

Für a > b sind alle diese Mengen leer. Desweiteren definieren wir für gegebenes ε > 0, ε ∈ ℝ und x ∈ ℝ die ε-Umgebung von x als

Uε(x) :=]x − ε,x + ε[= {y ∈ ℝ||x − y| < ε}.

Diese Umgebungen wachsen offensichtlich mit ε, d.h. für gegebenes x ∈ ℝ gilt

Uε1(x) ⊂ Uε2(x)fürε1 ≤ ε2.
