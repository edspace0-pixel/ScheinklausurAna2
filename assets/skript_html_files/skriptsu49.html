2.1.2  Grenzwerte reeller Folgen.

Wir betrachten nun eine Folge {xn}n∈ℕ reeller Zahlen xj ∈ ℝ und definieren den Begriff des Grenzwertes in ℝ:

Definition 2.1.1. Die Folge {xn}n∈ℕ konvergiert für n →∞ gegen den Grenzwert a ∈ ℝ, genau dann wenn folgende Aussage wahr ist:

∀ε>0,ε∈ℝ∃Nε∈ℕ∀n∈ℕ,n≥Nεxn ∈ Uε(a).

Man schreibt dann1 lim n→∞xn = a bzw. xn→n →∞a.

Remark 2.1.2. Für rationale Folgen xn ∈ ℚ, n ∈ ℕ und rationale Grenzwerte a stimmt diese Definition mit jener der Konvergenz in ℚ überein: Zu jedem ε > 0, ε ∈ ℝ existiert ja ein ε̃ ∈ ℚ mit 0 < ε̃ < ε < 2ε̃, damit sind die Definitionen 2.1.1 und 1.7.3 für eine solche Folge gleichzeitig erfüllt.

 

Remark 2.1.3. Der Grenzwert einer konvergenten Folge ist eindeutig bestimmt: Man kann leicht den Beweis von Satz 1.7.5 für reelle Grenzwerte reeller Folgen verallgemeinern..

Example 2.1.4. Es sei {xn}n∈ℕ = {c}ℕ die konstante Folge. Dann gilt offensichtlich xn ∈ Uε(c) für alle n ∈ ℕ und ε > 0 und damit c = lim n→∞c, die konstante Folge konvergiert also.

1Wiederum beinhaltet diese Notation sowohl das Faktum der Konvergenz der Folge als auch den Wert des Grenzwertes.
