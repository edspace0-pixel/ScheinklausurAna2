2.2.3  Der Beweis des Satzes von Cauchy - Vorbereitungen.

Lemma 2.2.11. Es sei {rk}k∈ℕ ∈ CF(ℚ), x ∈ ℝ und {rk}k∈ℕ ∈ x. Dann folgt {rk+n0}k∈ℕ ∈ x für  alle n0 ∈ ℕ, (2.11) {|rk|}k∈ℕ ∈|x|. (2.12)

Beweis. Zu (2.11): Offensichtlich ist {rk+n0}∈ CF(ℚ). Weiterhin gilt lim k→∞(rk − rk+n0) = 0: Für jedes ε > 0 existiert ja ein Nε mit |rm − rn| < ε für n,m ≥ Nε. Setzt man m = n + n0 so ergibt sich gerade |rn − rn+n0| < ε für n ≥ Nε. Damit erhalten wir {rk}∼ {rk+n0} und die beiden Folgen gehren also der gleichen Äquivalenzklasse x an.

Zu (2.12): Wegen ||rn|−|rm||≤|rn − rm| gilt zunächst {|rn|}n∈ℕ ∈ CF(ℚ) für {rn}n∈ℕ ∈ x. Wir unterscheiden nun drei Fälle:

Fall 1: x = 0. Dann gilt rn → 0, d.h. |rn − 0| = |rn| = ||rn|− 0| < ε für grosse n und damit auch |rn|→ 0, also {|rn|}n∈ℕ ∈ 0 = |0|.

Fall 2: x > 0. Nach Definition der Ordnung gibt es für {rn}n∈ℕ ∈ x zwei rationale Zahlen ϵ1,ϵ2 und n0 ∈ ℕ mit 0 < ϵ1 < ϵ2 < rn für n ≥ n0. Damit ist nach (2.11)

{rn}n∈ℕ ∼{rn+n0}n∈ℕ = {|rn+n0|}n∈ℕ ∈ x = |x|.

Fall 3: x < 0. Analog zu Fall 2. Bearbeiten Sie diese Situation selbständig! □

Lemma 2.2.12. Es sei {rn}n∈ℕ ∈ CF(ℚ), x ∈ ℝ und {rn}n∈ℕ ∈ x. Dann folgt5

x= (ℝ,d|⋅|) lim n→∞rn.

Beweis. Aus {rn}n∈ℕ ∈ x und {rm,rm,rm,…}∈ rm folgt aufgrund der Definition der Addition und der Multiplikation in ℝ

{rn − rm}n∈ℕ ∈ x − rm

und damit nach (2.12) auch {|rn − rm|}n∈ℕ ∈|x − rm| . Wegen {rn}n∈ℕ ∈ CF(ℚ) gibt es für jedes ε > 0, ε ∈ ℚ ein Nε∕4 ∈ ℕ mit |rn − rm| < ε 4 < ε 2 < ε für alle n,m ≥ Nε∕4, und damit nach Definition von Ungleichungen in ℝ schliesslich |x − rm| < ε für beliebiges m ≥ Nε∕4. Letzteres bedeutet rm → x in (ℝ,d|⋅|). □

5Hier identifizieren wir wie vereinbart eine rationale Zahl rk in ℝ mit der ihr zugehörigen Äquivalenzklasse der konstanten Folge [{rk,rk,rk,…}]∼.
