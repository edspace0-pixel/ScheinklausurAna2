2.7.1  Die Struktur des komplexen linearen Vektorraumes.

Eine Menge X nennt man einen komplexen linearen Vektorraum, wenn auf X eine Addition + : X × X → X sowie die Multiplikation von Elementen aus X mit komplexen Zahlen ⋅ : ℂ × X → X definiert sind, welche die analogen Eigenschaften zu (2.22) erfüllen, wobei die Quantoren ∀α∈ℝ und ∀α,β∈ℝ durch ∀α∈ℂ und ∀α,β∈ℂ zu ersetzen sind.
