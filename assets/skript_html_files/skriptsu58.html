2.2.5  Die Existenz von Grenzwerten beschränkter monotoner reeller Folgen: Eine Anwendung des Satzes von Cauchy.

Definition 2.2.13. Eine Folge {xn}n∈ℕ reeller Zahlen heißt monoton  wachsend({xn}↑) ⇔def ∀n∈ℕ(xn ≤ xn+1), streng  monoton  wachsend({xn}↑↑) ⇔def ∀n∈ℕ(xn < xn+1), monoton  fallend({xn}↓) ⇔def ∀n∈ℕ(xn ≥ xn+1), streng  monoton  fallend({xn}↓↓) ⇔def ∀n∈ℕ(xn > xn+1).

Theorem 2.2.14. Es sei {xn}n∈ℕ eine monoton wachsende, beschränkte Folge reeller Zahlen. Dann konvergiert diese Folge in (ℝ,d|⋅|) gegen einen Grenzwert x ∈ ℝ.

Beweis. Wir zeigen, dass {xn}n∈ℕ ∈ CF(ℝ). Dann folgt aus dem Satz von Cauchy die Existenz eines Grenzwertes. Angenommen {xn}n∈ℕ∉CF(ℝ). Dann gilt nach formaler Negation von (2.10)

∃ε0>0∀N∈ℕ∃m,n≥N|xm − xn|≥ ε0. (2.17)

Wähle nun N1 = 1. Dann gilt aufgrund der Monotonie und (2.17) xm1 ≥ xn1 + ε0 ≥ x1 + ε0 für gewisse m1 > n1 ≥ 1. Wir setzen nun iterativ Nk = max{mk−1,nk−1} und finden wegen (2.17) Folgenelemente

xmk ≥ xnk + ε0 ≥ xNk−1 + ε0 ≥ x1 + (k − 1)ε0.

Diese Folge ist aber wegen ε > 0 klar unbeschränkt, was zum Widerspruch führt. □
