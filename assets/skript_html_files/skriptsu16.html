1.6.5  Eine Eigenschaft der Vergleichsrelationen auf den natürlichen Zahlen.

Problem 1.6.3. Im restlichen Teil dieser Sektion beweisen wir die folgende Aussage:

Theorem 1.6.4. Für zwei beliebige natürliche Zahlen m,n ∈ ℕ gilt immer genau einer der folgenden Fälle

m < n,m = nodern < m. (1.17)

Lemma 1.6.5. Jede natürliche Zahl n≠1 besitzt einen Vorgänger m, d.h.

∀n∈ℕ,n≠1∃m∈ℕ(m′ = n).

Beweis. Wir betrachten die Menge M = {1}∪{n ∈ ℕ|∃m∈ℕ(m′ = n)}. IA: Offensichtlich gilt 1 ∈ M. IS: Mit n ∈ M gilt immer auch n′ ∈ M, denn n′ ist ja Nachfolger von m = n. Daraus folgt dann M = ℕ. □

Lemma 1.6.6. ∀n∈ℕ,n≠1(n > 1).

Beweis. Nach dem vorhergehenden Lemma gibt es einen Vorgänger m zu n = m′ und damit

n = m′= (1.8)m + 1= (1.13)1 + p

mit p = m und folglich n > 1. □

Lemma 1.6.7. ∀n,p∈ℕ(n + p≠n).

Beweis. Wir führen eine Induktion in n durch. IA: 1 + p= (1.13)p + 1 = p′≠1, da nach (P3) 1 nicht Nachfolger einer anderen natürlichen Zahl ist. IS: Es sei n + p≠n für beliebige p ∈ ℕ und gewisses n, woraus n′ + p≠n′ für beliebige p ∈ ℕ zu schliessen ist. GA: Es sei n′ + p = n′ für ein p ∈ ℕ. Dann gilt

n′ = (n + 1) + p = (n + p) + 1 = (n + p)′

und nach (P4) n = n + p, was zum Widerspruch führt. □

Wir beweisen nun Satz 1.6.4.

Beweis. Es sei n eine beliebige natürliche Zahl, welche wir fixieren. Wir betrachten die Mengen ℕ< := {k ∈ ℕ|k < n}, ℕ> := {k ∈ ℕ|n < k}, M := ℕ< ∪{n}∪ ℕ>.

Im Schritt 1 zeigen wir zunächst M = ℕ, damit tritt mindestens einer der Fälle (1.17) auf. IA:Es gilt 1 ∈ M. Für n = 1 ist dies klar, falls n≠1 so haben wir nach Lemma 1.6.6 1 ∈ ℕ< ⊂ M. IS: Es sei m ∈ M, wir zeigen dass dann m′ ∈ M. Dazu unterscheiden wir drei Fälle: Falls m = n so gilt m′ = n + 1 > n und m′ ∈ ℕ >. Falls m ∈ ℕ> so gilt m > n, also m = n + p und

m′ = (n + p)′= (1.8)(n + p) + 1= (1.12)n + (p + 1)= (1.8)n + p′.

Damit gilt m′ > n, d.h. m′ ∈ ℕ >. Im letzten Fall sei m ∈ ℕ< und damit n = m + p. Ist dabei n≠m′ so gilt p≠1. Dann gibt es nach Lemma 1.6.6 q ∈ ℕ mit p = q′ = q + 1. Wegen

n = m + (q + 1)= (1.13), (1.12)(m + 1) + q = m′ + q

folgt m′ < n und m′ ∈ ℕ <. Für n = m′ ist m′ ∈ M offensichtlich. Zusammenfassend folgt M = ℕ.

Im zweiten Schritt müssen wir beweisen, dass nur einer der drei Fälle (1.17) eintreten kann, d.h. ℕ< ∪{n}∪ ℕ> ist eine paarweise disjunkte Zerlegung von ℕ. Wir zeigen z.B. n∉ℕ<.Tatschlich, für beliebiges m ∈ ℕ< haben wir n = m + p≠m aufgrund Lemma 1.6.7. Die Untersuchung von n∉ℕ> und ℕ< ∩ ℕ> = ∅ überlassen wir dem Leser. □
