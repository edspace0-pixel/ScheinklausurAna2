3.2.1  Die Definition der Landau-Symbole.

Es sei (M,d) ein metrischer Raum, X ⊂ M und x0 ∈acc(X). Wir betrachten zwei Funktionen f,g : X → Kn mit K ∈{ℝ, ℂ} und n ∈ ℕ. Weiterhin sei ∥⋅∥ die Norm in Kn.

Definition 3.2.1. Man schreibt f= x → x0O(g) ⇔ ∃δ>0∃C∈ℝ∀x∈X∩Uδ(x0) ∥ f(x) ∥≤ C ∥ g(x) ∥, (3.4) f= x → x0o(g) ⇔ ∀ε>0∃δε>0∀x∈X∩Uδε(x0) ∥ f(x) ∥≤ ε ∥ g(x) ∥ . (3.5)

Diese Definition kann man auch auf den Fall übertragen, dass f und g Werte in verschiedenen normierten Räumen annehmen. Zudem lassen sich (analog zum Punkt 2.10.5) die Definitionen von o und O formal auf die Fälle x0 = ∞ oder x0 = ±∞ ausweiten.
