2.6.3  Die Struktur des reellen normierten Raumes.

Ein reeller linearer Vektorraum X ist ein reeller normierter Raum, wenn auf X eine Abbildung ∥⋅∥: X → ℝ definiert ist, welche die Norm auf X genannt wird und folgende Eigenschaften erfüllen muss

∀x∈X (∥ x ∥≥ 0) ∧((∥ x ∥= 0) ⇔ x = 0), ∀α∈ℝ∀x∈X ∥ αx ∥ =|α|∥ x ∥, ∀x,y∈X ∥ x + y ∥ ≤ ∥ x ∥ + ∥ y ∥ . (2.25)

Theorem 2.6.2. Auf einem reellen linearen Vektorraum X sei ein reelles Skalarprodukt gegeben. Dann definiert ∥ x ∥:= x, x eine Norm auf X, d.h. die Abbildung ∥⋅∥ erfüllt die Eigenschaften (2.25).

Im von uns hier hauptsächlich untersuchten Spezialfall X = ℝn ergibt sich die kanonische Definition der euklidischen Norm

∥ x ∥= x1 2 + … + xn 2,x = (x1,…,xn) ∈ ℝn. (2.26)

Zur Vorbereitung des Beweises von Satz 2.6.2 formulieren und beweisen wir zunächst die folgende wichtige Ungleichung von Cauchy-Schwarz-Bunjakowskij:

Lemma 2.6.3. Auf einem reellen linearen Vektorraum X sei ein reelles Skalarprodukt ⋅,⋅ gegeben. Dann gilt

∀x,y∈X|⟨x,y⟩|≤x, x ⋅y, y . (2.27)

Beweis. Es seien x,y ∈ X. Da für y = 0 aus der Linearität des Skalarproduktes im zweiten Argument x, 0 = 0 folgt, so ist (2.27) für y = 0 trivial. Es sei also y≠0 und damit y,y > 0. Nach (2.23) gilt für beliebiges λ ∈ ℝ 0 ≤ x + λy,x + λy = x,x + 2λ x,y + λ2 y,y.

Setzt man hier λ = −x,y y,y, so ergibt sich

0 ≤x,x −|x,y|2 y,y

und wegen y,y > 0 damit auch (2.27). □

Remark 2.6.4. Man sieht aus dem Beweis auch sofort, dass die Gleichheit in (2.27) genau dann auftreten kann wenn x + λy,x + λy = 0, also nach 2.23 genau dann wenn x + λy = 0, d.h. wenn x und y kollinear sind.

Beweis. Wir wenden uns nun dem Beweis von Satz 2.6.2 zu. Die erste Eigenschaft von (2.25) folgt direkt aus der dritten Eigenschaft von 2.23. Die zweite Eigenschaft von (2.25) folgt nach der zweiten Eigenschaft von (2.23) durch

∥ αx ∥2 = ⟨αx,αx⟩ = α2⟨x,x⟩ = |α|∥ x ∥2.

Die Dreiecksungleichung der Norm (dritte Eigenschaft in (2.23)) folgt aus dem Zusammenhang zwischen Skalarprodukt und Norm sowie (2.27) wegen ∥ x + y ∥2 = x + y,x + y = ∥ x ∥2+ ∥ y ∥2 + 2 x,y ≤ ∥ x ∥2+ ∥ y ∥2 + 2 ∥ x ∥∥ y ∥ ≤ (∥ x ∥ + ∥ y ∥)2.

Die Aussage von Problem 1.8.12 vervollständigt den Beweis. □
