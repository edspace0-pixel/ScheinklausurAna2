1.6.4  Herleitung elementarer Eigenschaften von Addition und Multiplikation

Die Addition der natürlichen Zahlen ist assoziativ und kommutativ: ∀n,m,k∈ℕ(n + m) + k = n + (m + k), (1.12) ∀n,m∈ℕn + m = m + n. (1.13)

Zur Illustration leiten wir (1.13) her, wobei (1.12) als bewiesen gilt:

Schritt 1:

∀m∈ℕm + 1 = 1 + m. (1.14)

Wir beweisen (1.14) durch vollständige Induktion in m. IA:Wegen 1 + 1 = 1 + 1 = 1′ = 2 gilt (1.14) für m = 1. IS: Es gelte (1.14) für ein bestimmtes m, wir müssen daraus m′ + 1 = 1 + m′ schliessen. Dies folgt aus

1 + m′= (1.9)(1 + m)′= (1.14)(m + 1)′= (1.8)(m′)′= (1.8)m′ + 1.

Schritt 2:

∀m,nm + n = n + m. (1.15)

Wir führen nun eine Induktion in n durch. Der IA für n = 1 folgt aus (1.14). Im IS setzen wir

∀m∈ℕm + n = n + m (1.16)

für gewisses n voraus und müssen daraus m + n′ = n′ + m für alle m ∈ ℕ schliessen. Dies folgt wiederum aus m + n′= (1.9) (m + n)′= (1.16) (n + m)′= (1.8) (n + m) + 1= (1.12) n + (m + 1)= (1.14) n + (1 + m)= (1.12) (n + 1) + m= (1.8) n′ + m.□
