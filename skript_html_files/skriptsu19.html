1.7.2  Folgen.

Definition 1.7.2. Sei A eine Menge. Eine Folge {an}n=1∞ = {a 1,a2,…} von Elementen an aus A ist eine Abbildung

f : ℕ → A,an = f(n).

Obwohl die Notation {an}n=1∞ dies eventuell nahelegt, so sind Folgen doch keine Mengen! Zum ersten kann ein und dasselbe Element aus A beliebig oft als Folgenglied auftreten, zum zweiten ist Reihenfolge der Folgenglieder wichtig!
