2.6  Der Euklidische Raum ℝn

Die Menge ℝn ist gegeben durch das n-fache kartesische Produkt

ℝn = ℝ ×⋯ × ℝ︸ nFaktoren.

Elemente aus ℝn sind Vektoren x = (x1,…,xn) mit n reellen Komponenten xj ∈ ℝ für j = 1,…,n. Wir können dann die Operation der Addition + : ℝn × ℝn → ℝn zweier Vektoren x = (x1,…,xn) ∈ ℝn und y = (y1,…,yn) ∈ ℝn definieren:

x + y = (x1 + y1,…,xn + yn).

Die Multiplikation ⋅ : ℝ × ℝn eines Skalars9α ∈ ℝ mit einem Vektor x = (x1,…,xn) ∈ ℝn definiert man wie folgt

αx = (αx1,…,αxn).

Damit werden diese Operationen auf die Addition und Multiplikation auf dem Körper ℝ zurückgeführt.
